Building Blocks

  • [] - access arrays/strings and object properties
  • () - call functions and avoid errors
  • {} - get NaN and "[object Object]"
  • + - append strings, cast things to numbers
  • ! - cast things to booleans

Booleans

  • 0 - +[]
  • true - ![]
  • false - !![]
  • 1 - +!![]
